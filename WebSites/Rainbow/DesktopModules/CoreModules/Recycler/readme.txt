Recycler Module

This module renders a list of Modules that have been 'deleted' from a Portal where
PortalSetting "USE_RECYCLER" is set to true.


Originally created By: John Bowen, 03 Dec 2004 
Initial testing/improvement By: Chris Farrell, 06 Dec 2004
More Mods/improvements: John Bowen, May 2005

INSTALL
1. Go to Admin all and to add module definition.
2. Point to install.xml install file
3. Add the module to a page
4. Use it! ;o)

Issues and Known problems:
- Tested with Rainbow version - RC5 (early Summer 2005)
